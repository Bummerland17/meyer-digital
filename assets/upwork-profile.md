4. Select your country and fill in basic info

### Phase 4: Add Your Projects (Service Listings)

Upwork calls these "Projects" (also called "Project Catalog" entries). They're fixed-price service packages clients can browse and buy directly.

**To create a Project:**
1. Go to your profile → scroll to **"My Project Catalog"** → click **"Create a Project"**
2. Fill in:
   - **Title** → paste from each service
   - **Category** → match to the service type
   - **Description** → paste the 300-word description
   - **What's included** → paste the bullet list
   - **Price** → enter as listed
   - **Delivery time** → as specified
   - **FAQs** → add all 3 Q&As
3. Add a cover image (Canva works great — simple, professional, on-brand)
4. Repeat for all 3 services

*Note: Project Catalog may require your profile to be approved first. Submit your profile and check back — it typically unlocks within 24–48 hours.*

### Phase 5: Profile Optimization (After Setup)

- **Profile photo:** Professional headshot, good lighting, plain background. Smile.
- **Availability badge:** Turn on "Available Now" to appear higher in search
- **Profile visibility:** Set to *Public*
- **English proficiency:** Select *Native or Bilingual* if applicable
- **Connect your LinkedIn** (optional but adds credibility)

---

### Phase 6: First Week Strategy

1. **Don't wait for inbound.** Browse job postings in your categories and submit 3–5 tailored proposals per day.
2. **Write custom proposals.** Reference their specific business. Don't paste a template — Upwork clients can smell it.
3. **Offer a small starter project** if a client seems interested but hesitant — a mini audit, a single landing page, a content sample.
4. **Your first job matters most.** Even if it's at a slight discount, getting a 5-star review is worth it. Reviews unlock visibility.
5. **Set "Connects" budget** — you'll use Upwork's paid "Connects" to apply to jobs. Budget ~$20–30/month to start.
