## Pinterest Setup Guide

### Step 1: Create Business Account
1. Go to **https://pinterest.com/business/create**
2. Use email: **hello@pantrymate.net**
3. Business name: **PantryMate**
4. Website: **pantrymate.net**
5. Country: Namibia (or US for broader reach)
6. Category: **Food & Drink**
7. Click **Create Account**

### Step 2: Set Up Boards (Create These First)
Before pinning, create these 4 boards:

| Board Name | Description | SEO Keywords |
|-----------|-------------|--------------|
| **Dinner Ideas & Easy Recipes** | Quick weeknight dinners using what you already have | dinner ideas, easy recipes, quick meals |
| **Meal Planning Tips** | Smart meal planning strategies for busy families | meal planning, meal prep, weekly meals |
| **Zero Waste Cooking** | Use every ingredient, waste nothing | food waste, zero waste, frugal cooking |
| **What to Cook With [Ingredient]** | Ingredient-based cooking inspiration | what to cook with chicken, egg recipes, pantry cooking |

### Step 3: Apply for Pinterest API
1. Go to **https://developers.pinterest.com/apps/**
2. Log in with your business Pinterest account
3. Click **Create app**
4. App name: **PantryMate Scheduler**
5. App description: "Social media scheduling for PantryMate food app"
6. Redirect URI: `http://103.98.214.106:4007/integrations/social/pinterest/callback`
7. Submit — Pinterest API approval is typically **instant or same-day**

### Step 4: Connect to Postiz
1. Copy the **App ID** and **App Secret** from developers.pinterest.com
2. Go to Postiz → Settings → Social Media Credentials
3. Enter PINTEREST_CLIENT_ID and PINTEREST_CLIENT_SECRET
4. Restart Postiz: `docker-compose restart`
5. Go to Channels → Add Channel → Pinterest → Connect ✅

8. "pantry recipes" — niche but highly relevant

### Pin Optimization Checklist
- ✅ Include target keyword in **Pin title** (first 30 characters matter most)
- ✅ Include target keyword in **first sentence** of description
- ✅ Include pantrymate.net URL in every pin description
- ✅ Use **5–10 relevant hashtags** per pin
- ✅ Post to the most relevant board
- ✅ Use **vertical images** (2:3 ratio, 1000x1500px ideal)
- ✅ Warm, food-photography-style visuals perform best
- ✅ Text overlay on image helps (e.g., "5 Weeknight Dinners" as headline)

### Posting Frequency
- **Optimal:** 5–15 pins per day (Pinterest rewards frequent posting)
- **Minimum:** 2 pins per day (what Postiz is scheduled for)
- **Best times:** 8–11pm local time (high Pinterest usage)
