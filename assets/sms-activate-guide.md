## What Is This?

sms-activate.io is a service that provides real SIM-based phone numbers for one-time SMS verification. Unlike VoIP numbers (Google Voice, TextNow), these are real carrier numbers that bypass platform blocks on all major social networks.

**Total cost: ~$3–4** for all the numbers you need.

## Step 1: Create an Account & Add Balance

1. Go to **https://sms-activate.io**
2. Click **Register** (top right) — use any email
3. After registration, click **Top Up Balance**
4. Add **$5** (minimum) via:
   - **Cryptocurrency** (fastest — instant confirmation): Bitcoin, USDT, ETH, many others
   - **Credit/Debit Card** via PayPal or direct card
   - **Other options**: Payeer, Perfect Money, etc.
5. Balance shows instantly after payment

> 💡 **Pro tip:** Add $5 so you have buffer. Unused balance stays in your account forever.

## Step 2: Buy Numbers — Platform by Platform

### How to buy a number (same process for each platform):

1. In the left sidebar, search for the platform name (see exact names below)
2. Select **Country: United States** 🇺🇸 (best success rate)
3. Click the shopping cart icon next to the price
4. A phone number appears in your **Active Activations** panel
5. Use that number to create your account
6. Come back to sms-activate — the SMS code will appear within 1–5 minutes
7. Enter the code on the platform → ✅ Verified

## Pro Tips

### 🕵️ Incognito + Fresh IP
For each new account, use a fresh browser session:
- Open a new **Incognito/Private window** in Chrome or Firefox
- Ideally use a different IP for each account (mobile data vs WiFi, or VPN)
- This prevents platforms from flagging multiple accounts from the same browser/IP

### ⏰ Timing
- Numbers expire after **20 minutes** if no SMS arrives
- You can **request a new number** for free if the first one doesn't work (within the 20-min window)
- Most SMSes arrive within 1–3 minutes

### 🔄 If the Number Doesn't Work
- Click **Cancel** on the current number (you get a refund within 20 min)
- Try a different country (UK 🇬🇧 or Canada 🇨🇦 as backup)
- Or try a different number from the same country

### 📱 Keep Records
After each account is created, save in a secure place:
- Platform
- Email used
- Phone number used (for reference)
- Date created
