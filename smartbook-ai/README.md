## Deploy

### Netlify (drag & drop)
Drop the entire folder into [app.netlify.com/drop](https://app.netlify.com/drop)

### Netlify CLI
```bash
netlify deploy --prod --dir .
```

### GitHub Pages
Push to a GitHub repo and enable Pages in repository settings (root `/` or `main` branch).
