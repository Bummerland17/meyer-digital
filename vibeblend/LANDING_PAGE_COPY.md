## Objection-Handling / FAQ Section

### Section Headline
```
Real questions.
```

**Q: Do I need a Spotify Premium account?**  
```
A: Nope. VibeBlend works with any Spotify account — free or premium.  
We just need permission to read your listening history and create playlists.
```

**Q: Will VibeBlend post anything to my Spotify without asking?**  
```
A: Never. We only create playlists when you explicitly tap "Create Blend."  
That's it.
```

**Q: What if I don't like the blend?**  
```
A: Just delete the playlist from Spotify and blend again.  
On the free tier, each blend counts as your monthly one — so think of it like a cocktail: you can always try a different one next month.
```

**Q: Does VibeBlend work with Apple Music?**  
```
A: Not yet — Spotify's API gives us the audio data we need to match your taste.  
Apple Music integration is on our roadmap.
```

**Q: What data do you store?**  
```
A: Your Spotify user ID, display name, and a secure token to make API calls on your behalf.  
We don't store your listening history or track lists.
```
